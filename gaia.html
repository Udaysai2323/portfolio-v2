Major MCA Project

Python OpenCV MediaPipe Speech Recognition AI

Project Overview

G.A.I.A (Gestural and Auditory Intelligent Assistant) is an AI-powered desktop assistant that enables natural human–computer interaction using voice commands and hand gestures.

The system supports intelligent automation, emotional interaction modes such as Roast Mode and Therapist Mode, and real-time system control with response times under 2 seconds.

Key Features

Voice Control

Execute commands, play music, search, translate languages using speech.

Gesture Control

Control music and applications using real-time hand gestures via webcam.

Emotional Intelligence

Human-like responses, roast mode, therapist mode, and sarcasm.

System Automation

Open apps, control volume & brightness, screenshots, automation tasks.
